Prerequisites for running the quality system.
===

1. Python 2.7 - http://python.org
2. Virtual env and pip for Python.

Installing the software
===

$HQ_VIRTUAL_ENV=/home/nfsusers/jenkins/hq-pyenv  # For example

# Delete previous version
if [ -d $HQ_VIRTUAL_ENV ]; then
    rm -rf $HQ_VIRTUAL_ENV
fi

# Create virtualenv and activate it
virtualenv $HQ_VIRTUAL_ENV
. $HQ_VIRTUAL_ENV/bin/activate

# Install HQ
pip install quality_report
