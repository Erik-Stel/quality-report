HQ Refresh Token Generator


Using this tool you will generate a refresh token for accessing azure Sharepoint services. Refresh token is long living token used to perpetually acquire authentication tokens and enable HQ to connect and fetch data from Sharepoint services.

First, you should register your application at https://portal.azure.com. You will need the Application ID and ClientSecret from there. Also, you have to set Reply URL to http://localhost:5005/login/authorized.
